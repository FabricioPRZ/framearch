## [Unreleased]

### Added

- Initial CLI with interactive prompts (`@inquirer/prompts`)
- Framework registry: React, Vue 3, Svelte, Angular
- Architecture registry with WIP flag support
- Screaming Architecture — full templates for all 4 frameworks
- MVC architecture stub (WIP)
- MVVM architecture stub (WIP)
- Dry-run mode (preview files without writing)
- `previewGenerator` pure helper for testing
- Vitest test suite with coverage thresholds
- CI workflow (Node 18/20/22, lint, typecheck, test)
- Publish workflow (npm + GitHub Release on `v*` tags)
- Issue templates (bug report, feature request)
- Pull Request template with architecture/framework checklists
- CONTRIBUTING.md guide
